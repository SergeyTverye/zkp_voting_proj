For the project to work it is necessary to have node.js stable version and MySQL server installed on PC. 

MySQL configuration is set at the beginning of index.js files for both servers. 

The description of the required tables for MySQL can be found in the database.sql file

To start the project you need to go to the directory: /backend/main-server
execute the command: 
node index

In the directory: /backend/voiting-server/
execute command: 
node index

In the directory : /frontend
execute the commands:
npm install
npm start
